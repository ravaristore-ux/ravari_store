──────────────────────────────────────

Add 3 secrets:
1. HOSTINGER_USER = your_username
2. HOSTINGER_HOST = your_host.hostingersitecom
3. HOSTINGER_PASSWORD = your_password

📞 SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues:

1. "Site not deploying"
   → Check GitHub Actions logs
   → Verify SSH credentials in secrets
   → Check Hostinger status

2. "API returns 404"
   → Verify backend is running
   → Check reverse proxy (.htaccess)
   → View error logs

3. "Database connection failed"
   → Check MongoDB Atlas connection string
   → Verify IP whitelist
   → Test connection manually

4. "Frontend shows blank page"
   → Check browser console (F12)
   → Verify REACT_APP_API_URL
   → Run npm run build
